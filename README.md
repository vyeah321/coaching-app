# Coaching App

コーチング関連のプロトタイプアプリケーション集

## プロジェクト一覧

### [session_organizer](./session_organizer/)
コーチがセッション後に行う「整理・振り返り・次回準備」を短時間で完了できるアプリ

**機能:**
- セッションメモからレポート自動生成
- クライアント向けレポート
- コーチ向けメモ
- Google Drive 連携

**技術:** Streamlit, Python, Google Drive API
