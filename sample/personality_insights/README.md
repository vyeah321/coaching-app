# 🎯 パーソナリティインサイト診断アプリ

あなたの強み、コミュニケーションスタイル、リーダーシップタイプを発見する診断アプリです。

## ✨ 特徴

### 🎯 3種類の診断

#### 1. 💪 強み診断
あなたの持っている5つの才能を発見します。
- **分析力**: 論理的思考と問題解決能力
- **創造力**: 新しいアイデアと革新的発想
- **共感力**: 人の気持ちの理解と関係構築
- **実行力**: 計画の実行と確実な成果
- **リーダーシップ**: チームをまとめる力

**診断時間**: 約3分（15問）

#### 2. 💬 コミュニケーションスタイル診断
あなたのコミュニケーションの特徴を分析します。
- **論理型**: 事実とデータを重視する
- **感情型**: 感情表現が豊かで共感的
- **行動型**: 効率重視で結果を求める
- **観察型**: 慎重に全体を見て判断

**診断時間**: 約2分（12問）

#### 3. 👑 リーダーシップスタイル診断
あなたのリーダーシップの型を判定します。
- **ビジョン型**: 大きな目標で人を鼓舞
- **コーチ型**: メンバーの成長を重視
- **民主型**: 合意形成とチームワーク
- **ペースセッター型**: 自ら率先して引っ張る

**診断時間**: 約2分（12問）

## 📊 機能

### 診断機能
- 5段階評価の質問形式
- 進捗バー表示で残り時間が分かる
- 前の質問に戻ることも可能

### 結果表示
- **レーダーチャート**: 視覚的にスコアを表示
- **詳細スコア**: 各カテゴリの点数（0-100点）
- **パーソナライズドアドバイス**: 最も高いスコアに基づいた具体的なアドバイス
- **強みの活かし方**: キャリアや仕事での活用方法

### 履歴機能
- 過去の診断結果を保存
- いつでも振り返り可能
- 複数回受けて変化を確認

## 🚀 セットアップ

### 必要な環境
- Python 3.8以上

### インストール手順

1. 依存パッケージのインストール:
```bash
pip install -r requirements.txt
```

2. アプリケーションの起動:
```bash
streamlit run app.py
```

3. ブラウザで自動的に開かない場合は、表示されるURLにアクセス
   - 通常は http://localhost:8501

## 📁 データの保存

診断結果は `results/diagnosis_results.json` に自動保存されます。

## 💡 使い方

### 診断を受ける

1. トップページで3つの診断から1つを選択
2. 各質問に5段階で回答
   - 1: 全くそう思わない
   - 2: あまりそう思わない
   - 3: どちらとも言えない
   - 4: ややそう思う
   - 5: 非常にそう思う
3. すべての質問に回答すると結果が表示されます

### 結果の見方

- **レーダーチャート**: 各カテゴリのバランスが一目で分かります
- **スコア**: 70点以上が強み、50-69点は中程度、50点未満は発展途上
- **最高スコア**: あなたの最大の強みとして詳細アドバイスが表示されます

### 診断履歴

サイドバーの「診断履歴」から過去の結果を確認できます。

### 個人での活用
- 自己理解を深める
- キャリアの方向性を考える
- 強みを活かした目標設定

### コーチングでの活用

- 話題のきっかけ作り

- 定期的な診断で成長を可視化

### チームでの活用

- 役割分担の参考に
- コミュニケーション改善のヒント

## 🔧 カスタマイズ

### 質問の追加・変更
`app.py` の `QUESTIONS` 辞書を編集してください。

### アドバイスの変更
`app.py` の `ADVICE` 辞書を編集してください。

### デザインの変更
カスタムCSSセクションでスタイルを調整できます。

## ⚠️ 注意事項

- この診断は自己理解のためのツールです
- 専門的な心理検査ではありません
- 結果は参考として、コーチとの対話を深めるために活用してください
- 何度でも受けられますが、正直に答えることが大切です

## 🌟 おすすめの使い方

1. **初回**: すべての診断を一度受けてみる
2. **振り返り**: 結果をコーチと一緒に見ながらセッション
3. **定期診断**: 3ヶ月ごとに再診断して変化を確認
4. **目標設定**: 診断結果を元に強みを活かした目標を立てる
