# VAK学習スタイル診断アプリ 🎯

Visual（視覚型）、Auditory（聴覚型）、Kinesthetic（体感覚型）の学習スタイルを診断するStreamlitアプリです。

## 概要

VAK学習スタイルモデルに基づいて、個人の情報処理や学習の好みを診断します。30の質問に答えることで、あなたの優勢な学習スタイルを特定し、効果的な学習方法を提案します。

## 機能

### 1. 診断機能
- 30問の質問（各タイプ10問ずつ）
- 5段階評価での回答
- 進捗バーで現在地を表示
- 前の質問に戻る機能

### 2. 結果表示
- レーダーチャートでの可視化
- 各タイプのスコア表示
- 優勢な学習スタイルの判定
- 各スタイルの詳細説明

### 3. 学習アドバイス
- 各学習スタイルの強み
- 効果的な学習のヒント
- 総合的なアドバイス
- バランス型の診断

### 4. 履歴管理
- 過去の診断結果を保存
- 履歴の閲覧機能

## インストール

```bash
pip install -r requirements.txt
```

## 実行方法

```bash
streamlit run app.py
```

## VAK学習スタイルとは

### Visual（視覚型）📊
- 視覚的な情報処理が得意
- 図、グラフ、画像で理解しやすい
- 色やレイアウトに敏感

### Auditory（聴覚型）🎵
- 聴覚的な情報処理が得意
- 話を聞いて理解するのが得意
- ディスカッションで理解が深まる

### Kinesthetic（体感覚型）🤸
- 身体を使った学習が得意
- 実践的な学習が効果的
- 体験から学ぶのが得意

## データ保存

診断結果は `results/vak_results.json` に保存されます。

## ユースケース

- **個人**: 自分の学習スタイルを理解し、効率的な学習方法を発見
- **教育者**: 生徒の学習スタイルを把握し、適切な指導方法を選択
- **コーチ**: クライアントの特性を理解し、コミュニケーション方法を最適化
- **組織**: チームメンバーの特性を理解し、効果的な情報共有方法を構築
