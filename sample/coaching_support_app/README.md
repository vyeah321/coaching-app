# コーチングサポートアプリ

コーチングセッションを効率的に管理するためのStreamlitアプリケーションです。

## 🎯 主な機能

### 1. クライアント管理
- クライアント情報の登録・管理
- 連絡先情報の保存
- メモの記録

### 2. セッション記録
- セッションごとの詳細な記録
- テーマ、概要、気づきの記録
- アクションアイテムの管理

### 3. 目標トラッキング
- クライアントごとの目標設定
- 進捗状況の可視化
- カテゴリ別の目標管理
- 期限の設定と管理

### 4. レポート
- 統計情報の表示
- クライアント別の進捗レポート
- セッション時間の集計

## 🚀 セットアップ

### 必要な環境
- Python 3.8以上

### インストール手順

1. 依存パッケージのインストール:
```bash
pip install -r requirements.txt
```

2. アプリケーションの起動:
```bash
streamlit run app.py
```

3. ブラウザで自動的に開かない場合は、表示されるURLにアクセスしてください
   - 通常は http://localhost:8501

## 📁 データの保存

アプリケーションは自動的に `data` ディレクトリを作成し、以下のJSONファイルにデータを保存します：

- `data/clients.json` - クライアント情報
- `data/sessions.json` - セッション記録

## 💡 使い方

### クライアントの登録
1. サイドバーから「クライアント管理」を選択
2. 「新規クライアント追加」を展開
3. 必要な情報を入力して「クライアントを追加」をクリック

### セッションの記録
1. サイドバーから「セッション記録」を選択
2. クライアントを選択
3. セッション情報を入力して記録

### 目標の設定と追跡
1. サイドバーから「目標トラッキング」を選択
2. クライアントを選択
3. 目標を追加し、進捗を更新

### レポートの確認
1. サイドバーから「レポート」を選択
2. 全体統計やクライアント別の詳細を確認

## 🛠️ カスタマイズ

### データディレクトリの変更
`app.py` の以下の行を編集してください:
```python
DATA_DIR = Path("data")  # お好みのパスに変更
```
