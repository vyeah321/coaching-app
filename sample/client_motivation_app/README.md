# 🌟 クライアント向けモチベーションアプリ

コーチングクライアントが目標達成を楽しみながら追跡できる、ワクワクするモチベーションアプリです！

## ✨ 特徴

### 🎮 ゲーミフィケーション要素
- **レベルシステム**: 活動するほど経験値が貯まり、レベルアップ！
- **バッジ獲得**: 様々な達成条件でバッジをコレクション
- **連続記録**: 毎日の振り返りで連続日数を更新
- **ポイント制度**: すべての活動でポイントを獲得

### 🎯 主な機能

#### 1. ダッシュボード
- 現在のレベルと経験値の表示
- 今日のモチベーショナルメッセージ
- 今週の注目目標
- クイックアクション

#### 2. 目標設定
- カラフルな絵文字で目標を個性的に
- カテゴリ別管理（健康、学習、キャリアなど）
- 重要度と期限の設定
- リアルタイムの進捗トラッキング
- 目標達成時にバルーンでお祝い 🎉

#### 3. 今日の振り返り
- 毎日の気分トラッキング
- 達成したことの記録
- 学びと気づきの保存
- 感謝の記録
- 明日の計画

#### 4. 達成バッジ
- 獲得済みバッジの表示
- 未獲得バッジの確認
- 各バッジの獲得条件

#### 5. 設定
- プロフィール管理
- 統計情報の確認
- データ管理

## 🎁 獲得できるバッジ

- 🎯 **はじめの一歩** - 最初の目標を設定
- 🏅 **目標マスター** - 5つ以上の目標を設定
- 🎉 **アチーバー** - 目標を達成
- 📝 **振り返りビギナー** - 初めての振り返り
- 🔥 **7日連続** - 7日間連続で振り返り
- 🔥🔥 **30日連続** - 30日間連続で振り返り

## 🚀 セットアップ

### 必要な環境
- Python 3.8以上

### インストール手順

1. 依存パッケージのインストール:
```bash
pip install -r requirements.txt
```

2. アプリケーションの起動:
```bash
streamlit run app.py
```

3. ブラウザで自動的に開かない場合は、表示されるURLにアクセス
   - 通常は http://localhost:8501

## 📁 データの保存

アプリケーションは自動的に `client_data` ディレクトリを作成し、以下のファイルにデータを保存します：

- `user_profile.json` - ユーザープロフィール、レベル、経験値
- `goals.json` - 目標情報
- `achievements.json` - 獲得したバッジ
- `daily_log.json` - 毎日の振り返り記録

## 💡 使い方

### 初めての起動
1. アプリを起動すると、自動的にプロフィールが作成されます
2. サイドバーで「⚙️ 設定」から名前を設定しましょう

### 目標を設定する
1. 「🎯 目標設定」ページへ移動
2. 「新しい目標を追加」で目標を入力
3. 好きな絵文字とカテゴリを選択
4. 目標を設定すると経験値とバッジがもらえます！

### 進捗を更新する
1. 目標リストから目標を開く
2. スライダーで進捗を更新
3. 100%達成すると自動的に「達成」ステータスに！
4. バルーンとともにお祝いメッセージが表示されます 🎉

### 毎日の振り返り
1. 「📝 今日の振り返り」ページへ
2. 今日の気分や達成したことを記録
3. 感謝や学びを書き出す
4. 連続日数が増えると特別なバッジがもらえます！

### レベルアップのコツ
- 目標を設定: +10 ポイント
- 進捗を更新: 進捗に応じてポイント
- 目標達成: +50 ポイント
- 毎日の振り返り: +20 ポイント
- 100ポイントごとにレベルアップ！

## 🎨 カスタマイズ

### モチベーショナルメッセージの追加
`app.py` の `get_motivational_quote()` 関数内のリストに追加してください。

### 新しいバッジの追加
1. `all_badges` 辞書に新しいバッジを定義
2. `check_badge()` 関数を適切な場所で呼び出し
